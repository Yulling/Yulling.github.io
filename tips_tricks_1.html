Image Optimimization

To optimize my images, I used Photoshop. There is a feature in Photoshop under export called Save for Web (Lagacy) that has 4 types of image compression types. One of them is called Optimized. The Dahlia image was jpeg at 22MB. Using the optimize tool and resizing from W4000px H3989px to 12.5%. The image size dropped to 89KB and size is W400px H300px and is .gif.

The red panda image was originaly W4000px H3000px at 20MB. Using Photoshop Save for Web - Optimized, I was able to scale the image to 82KB .gif file at W400px H300px.
